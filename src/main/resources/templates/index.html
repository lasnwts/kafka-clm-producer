Service: zk-chd-cft

Сервис передачи файлов из ЗК в Банк, и из Банка в ЗК

Назначение приложения

Сервис получения сообщений из ab, cdb, pm, sca brk

(МикроСервис формирования файлов для Золотой Короны) МП https://sdportal/mprojects/958766. Инициатива: https://devsystem.smartsoftware.ru/initiatives/2289 Проект 5581.

Запуск программы.

 

Команда запуска приложения : Краткое : Описание (Версия: java 11.)

java -jar zk-chd-cft.jar

 Для управления проектом можно использовать внешний файл application.properties.
Файл должен быть расположен в одном каталоге с файлом Jar или подкаталоге ./config

 

 

Описание application.properties

 

Опции в файле : Наименование Краткое : Описание

server.port

 Порт на котором будет работать tomcat

management.endpoints.web.exposure.include

 Actuator property
Какие опции открыть для получения метрик в actuator'e

spring.banner.location

 Банер Уралсиба (просто украшение)

Log level

 Уровни логирования в приложении (INFO,WARN,DEBUG,TRACE)

logging.file.path

 Здесь задается полный путь куда записывать лог приложения 
spring.datasource
 Параметры по подключению к БД 
spring.kafka
 Праметры подключения к брокеру Кафка
